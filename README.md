# Проект: Corridor Chronicles
## Кратко описание
Това е 3D интерактивен софтуер (игрова платформа), разработен с игрови енджин (UNITY), която прецизно пресъздава реална училищна среда с висока степен на автентичност и детайлност. Архитектурата, коридорите, класните стаи и дворът са моделирани по реалния обект, като целта е пълно потапяне в позната, но оживена дигитална версия на училището.

Играчът свободно изследва пространството, взаимодейства с обекти и персонажи, и изпълнява разнообразни мисии – образователни, логически и приключенски. Играта съчетава реалистична 3D графика, динамично осветление и интуитивно управление, като превръща училището в интерактивна платформа за обучение, ориентация и забавление.

**Какъв проблем решава проектът?**
Проектът **решава проблема** с липсата на интерактивни и модерни средства за представяне и опознаване на училищната среда. Той улеснява ориентацията на нови ученици и гости, повишава ангажираността към училищния живот и показва образователната институция по атрактивен и технологично актуален начин.

# Изтегляне на Visual Studio 
1. Избор на версия
   - Отиди на официалния сайт: https://visualstudio.microsoft.com
   - Ще видиш няколко версии:
     - Community – безплатна, подходяща за студенти и индивидуални разработчици.
     - Professional – платена, за малки екипи.
     - Enterprise – платена, за големи компании с разширени функции.
   - За учебни и лични проекти избери Community.
2. Изтегляне
   - Натисни Free download под Community.
3. Инсталация
   - Стартирай изтегления файл.
   - Ще се отвори инсталационен прозорец, в който можеш да избереш Workloads (пакети за различни типове разработки).
За Unity препоръчително е да избереш:
     - Game development with Unity (включва C# и Unity tools)
     - Можеш да добавиш и .NET desktop development, ако ще пишеш C# програми извън Unity.
   - Натисни Install и изчакай инсталацията да завърши.
4. Стартиране и свързване с Unity
   - Стартирай Visual Studio.
   - Ако имаш Unity проект, той автоматично ще може да се използва като IDE за скриптове.
   - В Unity → Edit → Preferences → External Tools можеш да избереш Visual Studio като default IDE.
   - Изборът е твой какво ще пишеш на C#.
 
# Изтегляне на Unity Hub
1. Unity Hub е централното приложение за управление на версии на Unity, проекти и лицензиране.
- Отиди на официалния сайт на Unity: https://unity.com
- Натисни Download Unity Hub (Windows или macOS според твоята система).
- Инсталирай Unity Hub, като следваш инструкциите на инсталационния прозорец.
2. Създаване на акаунт
- Стартирай Unity Hub.
- Влез с Unity акаунт. Ако нямаш акаунт, можеш да създадеш безплатен.
3. Изтегляне на Unity Editor чрез Unity Hub
- В Unity Hub отиди на таба Installs.
- Натисни Add Editor.
- Избери версията на Unity, която искаш да инсталираш (най-новата версия е препоръчителна, но можеш да избереш и LTS – Long-Term Support версия за по-голяма стабилност). Версия на проекта Unity 2022.3 LTS (2022.3.62f3).
- Избери допълнителни модули, които ще са ти нужни:
  - Visual Studio (ако искаш интегриран IDE за програмиране на C# - задължително при нас).
- Натисни Install и изчакай изтеглянето.
4. Отваряне на проект
  - В Unity Hub отиди на таба Projects.
  - Избери и натисни името на играта.
  - И готово играта се отваря.

# Substance 3D for Unity

1. Отвори Package Manager
   - Unity → Window → Package Manager
   - В горния ляв ъгъл избери Unity Registry (за да виждаш официалните пакети на Unity).
2. Намери пакета Substance 3D
   - Потърси Substance 3D или просто Substance в полето за търсене.
   - Избери Adobe Substance 3D for Unity (пакета може да се казва точно така или само „Substance“).
3. Инсталиране
   - Натисни Install.
   - Изчакай Unity да свали проекта.
   - Натисни Import → бъди сигурен, че си на All → Import
   ⚠️ Забележка: При инсталация Unity може да изтегли и допълнителни зависимости – остави ги да се инсталират.
4. Проверка
   - След инсталация папката Assets\Adobe\Substance3DForUnity трябва да се появи в проекта.
   - В проекта вече ще имаш достъп до скриптове и типове като ImportFileMessage.
